# consumer_behaviour_analysis

📌 Project Overview

This project focuses on analyzing consumer purchasing behavior using Python, SQL, and Power BI. The goal of the project is to uncover meaningful business insights related to customer demographics, purchasing patterns, product preferences, marketing influence, and customer engagement.

The project demonstrates a complete end-to-end data analytics workflow:

Data Cleaning & Preprocessing using Python
Business Query Analysis using SQL
Interactive Dashboard Development using Power BI

🎯 Objectives
Analyze customer purchasing behavior
Identify high-value customer segments
Understand the impact of marketing and discounts
Compare online vs in-store purchasing behavior
Generate actionable business insights through visualization

🔍 Key Business Insights
Electronics generated the highest revenue among all categories.
High-income customers contributed the largest share of revenue.
Social media influence significantly impacted purchasing behavior.
Customer activity was balanced across online, in-store, and mixed channels.
Moderate discount sensitivity positively influenced spending behavior.
Customer satisfaction showed room for improvement despite strong revenue generation.
